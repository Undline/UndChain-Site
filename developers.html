Want to develop for UndChain?

UndChain is under development at this time; UndChain also has a developer fund that helps support new developers coming in. This allows developers to earn coins while they develop, this is done by using something called the bounty system. This system allows anyone to come in and contribute code and earn based on the value of what they provided. This also applies to documentation and graphics.

Core Development

This involves the fundamental protocols with UndChain. Things such as network communications, cryptographic protocols, standardizing packet types, defining user types (Chain Owner, Validator, Partner, Client) as well as how they are meant to interact on the network. We are also working on scaling via sub-netting the network to allow more Validators to spread the load.

Co-Chain Development

This portion of development is meant for co-chains which is how UndChain can expand it's functionality (like importing a module). The co-chains that are currently under development are using Python rather than using Pseudo (our Pythonic programming language). Those Co-Chains are:

  • Auction House - A digital asset trading platform
  • Pages - Decentralized websites
  • SQeeL - Decentralized database
  • Mimic - AI framework
  • Code Ledger - Software repository system
  • Smack - Social media framework

How to join?

We do not have an effective means of joining at this time other than reaching out via our social media contacts. Please feel free to DM, please be sure to introduce yourself and state what you would like to see or offer the project. Even if you believe it's small or if you know someone who might be interested. The goal is to have permissionless development using Code Ledger which will natively provide rewards based on the contribution you are providing.
